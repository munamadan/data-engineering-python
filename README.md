# Data Engineering in Python - DataCamp Notes

Personal notes from the DataCamp "Data Engineering in Python" certification track.

## Using with Obsidian

1. Clone this repository
2. Open the cloned folder as a vault in Obsidian
3. Navigate through topics using Obsidian's graph view or file explorer

## Contents

This repository contains notes covering the core concepts and skills from the Data Engineering in Python track, including:

- Data processing and manipulation with pandas
- Working with databases and SQL
- Building data pipelines
- Data cleaning and validation
- ETL processes
- Working with APIs
- Data storage solutions

## Structure

Notes are organized by subject area, corresponding to the courses in the certification track. Each file contains key concepts, code examples, and practical applications.
