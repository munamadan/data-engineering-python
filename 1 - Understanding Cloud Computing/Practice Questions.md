## Quick Study Reference

### Service Models

- **IaaS:** System Admins, Rent infrastructure
- **PaaS:** Developers, Development platform
- **SaaS:** End Users, Ready software

### Market Share (3-2-1)

- AWS: **31%** (2006)
- Azure: **24%** (2010)
- GCP: **11%** (2008)

### Data Services

|Type|AWS|Azure|GCP|
|---|---|---|---|
|Warehouse|Redshift|Synapse|BigQuery|
|Streaming|Kinesis|Stream Analytics|Dataflow|
|ML|SageMaker|Azure ML|AutoML|

### Case Studies (75-50-40)

- NerdWallet (AWS): **75%** cost ↓
- Ottawa Hospital (Azure): **50%** DR cost ↓
- Lush (GCP): **40%** hosting cost ↓

### GDPR Essentials

- **72 hours:** Breach notification
- **€20M or 4%:** Fine (whichever higher)
- **Personal data:** Name, email, IP, health, location

### Cloud Roles

- **Architect:** Design infrastructure
- **Engineer:** Build & maintain
- **DevOps:** Automate (IaC)
- **Security:** Protect data

### Deployment Models

- **Private:** Exclusive, controlled, expensive
- **Public:** Shared, easy, cost-effective
- **Hybrid:** Mix, flexible, cloud bursting
- **Multicloud:** Multiple providers, no lock-in

### Scaling

- **Vertical ↑:** Upgrade one machine
- **Horizontal →:** Add more machines

---

## Key Decision Trees

### Choose Service Model

- Need OS control? → **IaaS**
- Building apps? → **PaaS**
- Just using software? → **SaaS**

### Choose Deployment

- Need full control? → **Private**
- Quick start, low cost? → **Public**
- Need both? → **Hybrid**
- Avoid lock-in? → **Multicloud**

### Choose Provider

- Microsoft ecosystem? → **Azure**
- Most services/mature? → **AWS**
- Data/AI/multi-cloud? → **GCP**
