## Essential Concepts

### Data Type Comparison

|Type|Syntax|Mutable|Unique|Ordered|Indexed|
|---|---|---|---|---|---|
|List|`[]`|Yes|No|Yes|Yes|
|Dict|`{k:v}`|Yes|Keys|Yes|By key|
|Set|`{}`|Yes|Yes|No|No|
|Tuple|`()`|No|No|Yes|Yes|

### When to Use What

**List:** Need order, allow duplicates, changeable **Dictionary:** Key-value pairs, quick lookup **Set:** Unique values only, remove duplicates **Tuple:** Immutable, fixed data

### Loop Comparison

|Feature|For Loop|While Loop|
|---|---|---|
|Use|Known iterations|Until condition|
|Syntax|`for x in y:`|`while x:`|
|Risk|None|Infinite loop|
|Control|break|break|

### Logical Operators

```python
# and - ALL must be True
True and True   # True
True and False  # False

# or - AT LEAST ONE True
True or False   # True
False or False  # False

# not - Negates
not True        # False
not False       # True

# in - Membership
"a" in ["a", "b"]  # True
```
